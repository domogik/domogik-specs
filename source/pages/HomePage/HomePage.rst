Presentation
============

Domogik is a free home automation solution. It is based on the ((xplProcol|xPL protocol)) in order to easily use a lot of different technologies. 
Domogik is multi hosts and modulable. You can use as many user interfaces you want (actually there are two user interfaces : the web one and the `Android <https://market.android.com/details?id=org.panel&amp;feature=search\_result>`_ one).
