Presentation¶

Domogik is a free home automation solution. It is based on the ((xplProcol|xPL protocol)) in order to easily use a lot of different technologies. Domogik is multi hosts and modulable. You can use as many user interfaces you want (actually there are two user interfaces : the web one and the `Android <https://market.android.com/details?id=org.panel&amp;feature=search\_result>`_ one). <br /><br /><br /> ||{img fileId=343} | __The end user documentation has migrated to a new tool. It is now available on http://docs.domogik.org/domogik/dev/en/ .<br /><br />The plugins documentation is available from this page : http://repo.domogik.org/package/ .__<br /><br />This wiki will new be used only for the developpers : spefication, development notes, ...||
