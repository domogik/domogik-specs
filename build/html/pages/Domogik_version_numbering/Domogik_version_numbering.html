Domogik’s version numbering¶

Domogik versions have this format : X.Y.Z

  • X = Major version
  • Y = Minor version
  • Z = See below

Alpha versions¶

x.y.z-alpha1
x.y.z-alpha2
...

Beta versions¶

x.y.z-beta1
x.y.z-beta2
...

Release Candidate versions¶

x.y.z-rc1
x.y.z-rc2
...

Stable versions¶

x.y.z{CODE}
