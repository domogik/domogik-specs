The purpose of this plugin is to manage an mvhr (Mechanical Ventilation with Heat Recovery) using xpl. For now, it only calculate exchanger efficiency.

Prerequisites¶

Install the plugin¶

This plugin is available in the development sources. Follow this documentation to install them. Enable the plugin : sudo dmgenplug -f mvhr

Configure and start the plugin¶

Enter the name of your mvhr and the indoor, outdoor and insufflation sensors.

Add a device¶

Add a “Exchanger eff.” device in domogik, using the mvhr name as address.

Add a widget¶

You can now add a widget to watch the efficiency of the heat exchanger.
